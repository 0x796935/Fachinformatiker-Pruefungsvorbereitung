## VDI - Virtual Desktop Infrastructure

- Klassische Virtualisierung
- Server im eigenen Unternehmen
- Muss selbst verwaltet werden

## DaaS - Desktop as a Service

- Virtualisierung in der Cloud
- Voll gemanagter Desktop
