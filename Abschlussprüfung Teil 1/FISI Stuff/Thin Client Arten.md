# Thin Client Arten

## All in one Thin Client
Beinhaltet alle Dinge die man braucht:
- Monitor
- Lautsprecher
- Anschlüsse
- Mikrofon
- Kamera

## Zero Thin Client
- Besitzt gar keinen eigenen Speicher
- Stark beschränkte Funktionen
- Nicht ohne Server funktional
